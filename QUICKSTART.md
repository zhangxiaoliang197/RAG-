# 🚀 快速启动指南

## 5 分钟快速上手

### 第一步：安装依赖（1分钟）

**macOS / Linux:**
```bash
bash install.sh
```

**Windows:**
```cmd
install.bat
```

### 第二步：启动服务（30秒）

**macOS / Linux:**
```bash
bash start.sh
```

**Windows:**
```cmd
start.bat
```

**或手动启动：**
```bash
npm install  # 如果还没安装依赖
npm run dev  # 启动开发服务器
```

### 第三步：访问系统

打开浏览器访问：**http://localhost:5173**

### 第四步：配置系统（3分钟）

#### 4.1 配置 Oracle 数据库

1. 点击左侧菜单 **"数据表管理"**
2. 点击右上角 **"Oracle 配置"** 按钮
3. 填写连接信息：
   - 用户名：您的 Oracle 用户名
   - 密码：您的 Oracle 密码
   - 主机：Oracle 服务器地址
   - 端口：1521（默认）
   - SID：您的数据库 SID
4. 点击 **"测试连接"** 验证连接
5. 点击 **"保存"** 保存配置

#### 4.2 配置 AI 模型（推荐）

**使用 Ollama（本地部署）：**
1. 安装 Ollama：访问 https://ollama.ai/download
2. 下载模型：
   ```bash
   ollama serve  # 启动服务
   ollama pull llama3.1  # 下载模型
   ```
3. 在系统设置页面配置：
   - 提供商：Ollama
   - Ollama 地址：`http://localhost:11434`
   - 模型：`llama3.1`

**使用 OpenAI（云端）：**
1. 在系统设置页面配置：
   - 提供商：OpenAI
   - API Key：您的 OpenAI API Key
   - 模型：gpt-4

### 第五步：开始使用（1分钟）

#### 5.1 添加数据表

1. 在 **"数据表管理"** 页面
2. 点击 **"从 Oracle 加载表"**
3. 选择要导入的表
4. 编辑表和字段的中文描述
5. 点击 **"保存"**

#### 5.2 构建知识库（可选但推荐）

1. 进入 **"知识库"** 页面
2. 添加常用指标定义：
   - 切换到 **"指标知识"** 标签
   - 点击 **"添加指标"**
   - 填写指标信息
3. 添加 SQL 示例：
   - 切换到 **"SQL 示例"** 标签
   - 点击 **"添加示例"**
   - 填写问题和对应的 SQL

#### 5.3 智能分析

1. 进入 **"智能分析"** 页面
2. 输入您想查询的问题，例如：
   - "计算华东区域本月的销售额"
   - "查看各区域的订单数量"
   - "找出购买金额前10名的客户"
3. 点击 **"智能分析"** 按钮
4. 系统会：
   - 生成 SQL 语句
   - 执行查询
   - 展示结果
5. 您可以：
   - 编辑生成的 SQL
   - 重新执行
   - 复制 SQL 到其他地方使用

## ⚠️ 常见问题

### Q: 连接 Oracle 失败？
**A:** 检查：
- Oracle 服务是否正常运行
- 网络连接是否正常
- 用户名密码是否正确
- 防火墙是否开放端口

### Q: AI 不工作？
**A:** 检查：
- Ollama 是否已启动（`ollama serve`）
- 模型是否已下载（`ollama list`）
- 网络连接是否正常

### Q: 查询超时？
**A:** 尝试：
- 添加更具体的查询条件
- 减少查询的时间范围
- 检查 SQL 语法
